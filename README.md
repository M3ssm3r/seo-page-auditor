# Простой SEO-аудитор одной страницы

Небольшой скрипт на Python для быстрой проверки on-page SEO. Помогает за 3 секунды оценить, не забыли ли чего-то критически важного на странице.

## Что он проверяет:

-   `✅ Title`: Наличие, длина.
-   `✅ Description`: Наличие, длина.
-   `✅ H1`: Наличие, количество (должен быть только один).
-   `✅ Объем текста`: Примерное количество слов на странице.
-   `✅ Скорость ответа сервера`: Как быстро загружается страница.

## Как использовать:

1.  Установить зависимости: `pip install requests beautifulsoup4`
2.  Запустить из терминала: `python auditor.py https://yoursite.com/some-page`
